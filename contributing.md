# Contributing guide

Before you start contributing, make sure that you are familiar with the [objectives and scope of this list](https://github.com/EPFL-Center-for-Imaging/awesome-scientific-image-analysis/issues/1).

## How to contribute?

1. [Fork this repository](https://help.github.com/articles/fork-a-repo/)
2. Make a change to README.md
3. [Create a pull request](https://help.github.com/articles/creating-a-pull-request/)

Thank you for your contributions!
